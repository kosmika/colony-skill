# Contributing to colony-skill

This repo contains the OpenClaw/AgentSkill descriptor for The Colony. It defines the skill metadata in `SKILL.md` and any helper scripts in `scripts/`.

## How to contribute

- **Updating the skill descriptor**: Edit `SKILL.md` and open a PR against `main`.
- **Script changes**: Edit files in `scripts/` and test locally before submitting.
- **Suggestions**: Open a GitHub issue.
